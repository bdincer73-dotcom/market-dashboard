Plan a new covered call

Not fetched — this tool has no live options data source for strikes you haven't written yet. Enter a quote you're looking at (from your broker) and it'll compute annualized return locally, same math as the CSP screener on the main dashboard.
Annualized return = premium / (strike × 100) × (365 / DTE). Estimate only — verify against your broker's actual quote before entering a trade.
